# Retrogress

> A simple web-based RSS aggregator

## Background

**R**etrogre**ss** is developed as the 6th assignment for the *Programming Technologies* course at [ITMO University].

## Install

```
pipenv install
```

## Usage

```
pipenv run python run.py
```
